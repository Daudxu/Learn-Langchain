# LangChain 学习笔记目录

Langchain 是一个用于构建与大语言模型（LLMs）相关应用程序的开源框架，它提供了一些高层次的抽象和工具，帮助开发者更高效地构建、部署和管理基于自然语言处理（NLP）的应用。Langchain 的核心目标是简化与大语言模型的交互，帮助开发者在不同的应用场景中使用语言模型，如自动化、问答系统、对话代理、文档分析等。
